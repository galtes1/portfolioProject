# Portfolio by Gal Testa

## Overview

This is a personal portfolio website created by Gal Testa, showcasing works and information about the developer. The project includes HTML and CSS files for styling and structure.

## Table of Contents

- [Features]
- [File Structure]
- [How to Use]


## Features

- Responsive design for various screen sizes.
- Stylish navigation bar and header.
- Cover section with a gradient background and animated text.
- About Me section with a brief introduction and an animated quote.
- CV section with download and view options.
- Gallery section with project cards.
- Contact section with a form and social icons.
- Mobile responsiveness for small screens.

## File Structure

- `index.html`: Main HTML file containing the structure of the portfolio.
- `style/`: Directory containing the CSS files.
  - `index.css`: Main stylesheet with styling for the entire project.
- `assets/`: Directory containing images, icons, and other assets.
- `projects/`: Directory containing project folders with individual HTML files and assets.

## How to Use

1. Clone or download the repository to your local machine.
2. Open the `index.html` file in a web browser to view the portfolio.
